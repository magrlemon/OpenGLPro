freeglut+vs（这里是vs2012） 使用说明：
//////////////////////////////////////////////////

将include中文件放到：
**\Microsoft**\Microsoft Visual Studio 11.0\VC\include

将bin中文件放到：
C:\Windows\SysWOW64			（对应x86中文件）	
C:\Windows\System32			（对应X64中文件）

将lib中文件放到：
**\Microsoft Visual Studio 11.0\VC\lib		（对应X64中文件）	
**\Microsoft Visual Studio 11.0\VC\lib\amd64	（对应x86中文件）
